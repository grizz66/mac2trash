# mac2trash

**Un-Apple your files.**  
mac2trash removes `.DS_Store`, `__MACOSX`, and other macOS junk files from directories and zip archives.

## Install

```bash
pip install .
```

## Use

```bash
mac2trash yourfolder/
mac2trash yourarchive.zip
```
